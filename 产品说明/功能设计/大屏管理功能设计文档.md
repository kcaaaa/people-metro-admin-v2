# 大屏管理功能设计文档

## 1. 功能概述
本模块主要负责管理系统中的各类可视化大屏，支持大屏的增删改查、实时预览及可视化编辑。本次更新主要针对界面UI进行调整，以匹配最新的视觉设计规范（参考Figma设计稿）。

## 2. 界面设计

### 2.1 头部区域
- **标题**: "大屏管理"
- **描述**: "集中展示APP直播列表信息，方便直播运营人员处理直播问题和建议。"
- **操作**: 右侧提供"同步数据"按钮，用于手动触发数据同步。

### 2.2 筛选区域
提供多维度筛选功能，支持以下字段：
- **大屏名称**: 文本输入
- **维护人**: 文本输入
- **维护时间**: 日期选择
- **创建人**: 文本输入
- **创建时间**: 日期选择

### 2.3 列表区域
表格展示大屏的详细信息，包含以下列：
- **序号**: 自动生成，补零显示（如001）
- **大屏名称**: 加粗显示
- **维护人**: 显示维护人员姓名
- **维护时间**: 格式化显示（yyyy-MM-dd HH:mm:ss）
- **创建人**: 显示创建人员姓名
- **创建时间**: 格式化显示（yyyy-MM-dd HH:mm:ss）
- **操作**:
  - **详情**: 查看大屏详情
  - **编辑**: 进入可视化编辑模式
  - **删除**: 删除该大屏配置

## 3. 交互逻辑

### 3.1 数据同步
- 点击"同步数据"按钮，显示"数据已同步"提示消息。

### 3.2 编辑大屏
- 点击列表中的"编辑"链接，进入大屏可视化编辑器（ScreenEditor）。
- 编辑器支持拖拽布局、模块内容修改、实时预览等功能。

### 3.3 删除大屏
- 点击"删除"链接，弹出二次确认对话框。
- 确认后删除对应的大屏数据，并刷新列表。

### 3.4 分页
- 支持自定义分页大小。
- 显示总记录数。

## 4. 数据结构
大屏对象（Screen）包含以下字段：
- `id`: 唯一标识
- `name`: 大屏名称
- `type`: 大屏类型（日常、协会版、标准版等）
- `status`: 运行状态（running, stopped, maintenance）
- `maintenanceUser`: 维护人
- `maintenanceTime`: 维护时间
- `createUser`: 创建人
- `createTime`: 创建时间
- `refreshInterval`: 刷新间隔（秒）

## 5. 样式规范
- **字体**: Microsoft YaHei
- **主色调**: #1890ff
- **背景色**: #f5f5f5 (页面背景), #ffffff (内容背景)
