人民城轨2.0是一个综合性的后台管理系统，本需求文档定义了系统的核心功能模块，包括情景管理、AI管理、文创管理、直播管理、审核管理等，旨在为企业提供全面的内容管理、数据展示和智能服务解决方案，提升企业的运营效率和服务水平?

### 2.2 智能体管理模?- **功能描述**：管理企业内部使用的AI智能体，支持完整的生命周期管理?- **核心功能**?  - 智能体增删改查操?  - 智能体启?关闭状态管?  - 按角色分配使用权?  - 知识库关联配?  - 智能体信息维护（名称、简介、模型选择、提示词配置?

## 3. 业务目标

- 提供集中式的大屏情景管理平台，实现大屏内容的智能化编排和自动播放
- 建立统一的AI资源管理中心，简化AI应用的运维复杂度
- 支持知识的结构化管理和复用，提升AI智能体的服务质量
- 实现细粒度的权限控制，保障系统资源的安全访问
- 提升企业内容管理和数据展示的效率，优化用户体?- 加速企业的数字化和智能化转型进?

- **系统管理?*：负责系统的整体管理和维护，包括用户权限配置
- **大屏运维人员**：负责情景的创建、编辑和管理，配置大屏播放内?- **内容编辑**：负责知识库内容和其他系统内容的上传和更?- **业务部门**：通过分配的权限使用系统功能和服务
