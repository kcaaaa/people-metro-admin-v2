## 🎯 5 分钟快速体验

### 步骤 1：打开编辑器
双击 `index.html` 文件，在浏览器中打开富文本编辑器。

### 步骤 2：输入文本
在左侧编辑器中输入一些文本内容，例如：
```
中国城市轨道交通协会章程

该章程共九章第81条，2023年2月6日经民政部核准生效...
```

### 步骤 3：插入图片
1. 点击工具栏的 📷 图片图标
2. 选择一张本地图片
3. 图片插入后，**自动弹出布局选择器**

### 步骤 4：选择图片布局 ⭐核心功能
布局选择器提供4种布局方式，每种都有可视化预览：
- **左对齐**：图片在左侧（45%宽度），文字环绕右侧
- **居中对齐**：图片居中显示（70%宽度）
- **右对齐**：图片在右侧（45%宽度），文字环绕左侧
- **全宽显示**：图片占满整行（100%宽度）

💡 **提示**：点击任意已插入的图片，可随时重新选择布局！

### 步骤 5：查看效果
右侧预览区域会实时显示编辑效果，所见即所得！

### 步骤 6：保存内容
点击底部的"保存内容"按钮，内容会保存到浏览器本地存储。

## 🔧 常见问题

### Q1: 图片太大怎么办？
**A:** 图片会自动适应容器宽度，但建议上传前先压缩图片。推荐使用在线工具如 TinyPNG 压缩图片。

### Q2: 如何删除图片？
**A:** 点击图片，然后按键盘的 `Delete` 或 `Backspace` 键即可删除。

### Q3: 如何更改已插入图片的布局？
**A:** 直接点击图片，会自动弹出布局选择器，选择新的布局方式即可。

### Q4: 文字环绕效果不明显？
**A:** 确保图片后面有足够的文字内容。如果文字太少，环绕效果不明显。左右对齐的图片宽度为45%，会为文字留出充足空间。

### Q5: 保存的内容在哪里？
**A:** 内容保存在浏览器的 localStorage 中。如果清除浏览器缓存，保存的内容会丢失。建议定期使用"导出 HTML"功能备份。

### Q6: 布局选择器可以关闭吗？
**A:** 可以。点击布局选择器右上角的关闭按钮（×）或点击遮罩层即可关闭，不选择任何布局。

### Q7: 可以插入网络图片吗？
**A:** 当前版本支持本地图片上传。如需使用网络图片，可以在代码中修改 `imageHandler` 函数。

### Q8: 如何在项目中使用？
**A:** 参考 `README.md` 中的"后续集成建议"章节，有详细的 Vue 3 集成示例代码。
