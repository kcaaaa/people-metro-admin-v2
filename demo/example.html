展示各种图片定位和文字环绕效果

一、图片左对齐 - 文字环绕右侧

示例 1：小图片左对齐

中国城市轨道交通协会章程（修正案）该章程共九章第81条，2023年2月6日经民政部核准生效。明确协会为全国性、行业性社会团体，住所设在北京，坚持党的全面领导，接受业务主管单位、行业管理部门的业务指导和监督管理。

章程规定协会业务范围广泛，通盖政策宣传、行业调研、行标行约制定、统计、人才培养、表彰建设、老年活动证、职称评审、行业活动举办、专业咨询、国际交流等。会员分为单位与个人会员，明确入会条件、程序、权利与义务、管理与章程。

组织机构包含会员大会（每5年召开，为权力机构）、理事会（执行机构、每届5年）、常务理事会（闭会期间行使部分职权）、负责人（设轮值会长、副会长、秘书长，执行秘书长等）、各机构权责与选任制新明确。

💡 实现方法：为图片添加 float: left 样式，并设置右侧和底部的外边距，文字会自动环绕在图片右侧。

二、图片右对齐 - 文字环绕左侧

示例 2：中等图片右对齐

中国城市轨道交通协会会制度汇编（2024版）该汇编由协会秘书处2024年1月发布，通盖25项管理制度，以"服务和自律"为核心，建协会治理逻辑与组织文化的依据。

内容上，会协会章程、会员管理相关规则约办法，还有业业业管理规则、装备产品认证审批公示，议事决策方面，有议事规则、会长常务会公会及监事会议事规则，明确会议召开、表决专机制、分委机构管理办法规范其设立、业务、财务与印章管理办法，还有各项专业工作管理上，会各等学术委员会、新闻发言人、团体标准、职业技能等多领域管理办法，通盖各委员会、标准修订、技能鉴定、出版、科技成果鉴定、科技进步奖、大型活动、媒体平台、职称评审、产品市场占有率证明等办法，规范各环节有序运行，保障协会工作，保障协会高效有序运行，助力行业发展。

💡 实现方法：为图片添加 float: right 样式，并设置左侧和底部的外边距，文字会自动环绕在图片左侧。

三、图片居中显示

示例 3：大图片居中

当图片需要作为重点展示内容时，可以使用居中对齐的方式。这种布局适合展示重要的图表、数据可视化或者关键场景图片。

此外，还有研究项、培训、出版、科技成果鉴定、科技进步奖、大型活动、媒体平台、职称评审、产品市场占有率证明等办法，规范各环节有序运行，保障协会工作，保障协会高效有序运行，助力行业发展。

💡 实现方法：为图片添加 display: block 和 margin: 0 auto 样式，图片会居中显示，不会有文字环绕效果。

四、混合排版示例

示例 4：多图片混合排版

轨道交通发展历程

中国城市轨道交通经历了从无到有、从小到大的发展历程。20世纪60年代，北京建成了中国第一条地铁线路，标志着中国城市轨道交通事业的开端。

进入21世纪后，随着城市化进程的加快和经济的快速发展，中国城市轨道交通进入了高速发展期。截至2023年底，全国已有超过50个城市开通了轨道交通，运营里程突破10000公里。

现代化的城市轨道交通系统不仅解决了城市交通拥堵问题，还成为了城市发展的重要基础设施。智能化、信息化、绿色化成为新时代轨道交通发展的主要方向。

未来，中国城市轨道交通将继续保持快速发展态势，预计到2030年，全国轨道交通运营里程将达到20000公里以上，为构建现代化综合交通运输体系提供有力支撑。

💡 实现方法：在同一段落中可以混合使用左对齐和右对齐的图片，创建丰富的图文混排效果。注意要在容器上添加 clearfix 类来清除浮动。

五、响应式图片排版

示例 5：自适应屏幕的图片

在大屏展示场景中，图片需要能够自适应不同的屏幕尺寸。通过设置 max-width: 100% 和 height: auto，图片可以在保持比例的同时适应容器宽度。

这种响应式设计确保了内容在各种显示设备上都能获得最佳的视觉效果，无论是大屏幕还是移动设备。

六、CSS 代码参考

核心样式代码
/* 图片左对齐 - 文字环绕右侧 */
.img-left {
    float: left;
    margin: 8px 24px 16px 0;
    max-width: 400px;
}

/* 图片右对齐 - 文字环绕左侧 */
.img-right {
    float: right;
    margin: 8px 0 16px 24px;
    max-width: 400px;
}

/* 图片居中 */
.img-center {
    display: block;
    margin: 24px auto;
    max-width: 600px;
}

/* 清除浮动 */
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}

/* 响应式图片 */
img {
    max-width: 100%;
    height: auto;
    border-radius: 8px;
}

七、使用建议

最佳实践
  • 图片尺寸：左右对齐的图片建议宽度不超过 400px，居中图片可以更大
  • 图片间距：图片与文字之间保持适当间距（16-24px），避免过于拥挤
  • 图片质量：使用高质量图片，但要注意文件大小，建议单张不超过 500KB
  • 图片比例：尽量使用 4:3 或 16:9 的常见比例，保持视觉协调
  • 清除浮动：使用浮动布局时，记得在容器上添加 clearfix 类清除浮动
  • 移动端适配：在小屏幕上，建议取消浮动，让图片独占一行
  • 无障碍访问：为所有图片添加有意义的 alt 属性
