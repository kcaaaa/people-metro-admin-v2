# 人民城轨管理系统 - 大屏管理模块全局UI设计文档

## 1. 设计概述

本文档详细描述人民城轨管理系统中大屏管理模块的UI设计规范、组件库使用、色彩方案、布局结构等内容，为前端开发提供统一的设计标准和参考。

## 2. 设计原则

### 2.1 总体设计原则

- **专业性**: 符合企业级应用的专业形象，界面简洁、清晰
- **易用性**: 操作流程直观，降低学习成本
- **一致性**: 保持UI元素的风格一致，提供统一的用户体验
- **可访问性**: 确保不同用户都能便捷地使用系统功能
- **可扩展性**: 设计可适应未来功能扩展和变化

### 2.2 大屏设计特殊原则

- **视觉冲击力**: 大屏展示需要强视觉冲击力，突出核心数据
- **清晰可读**: 即使在远距离观看，也能清晰识别信息
- **数据层次感**: 通过色彩、大小等元素，突出重要数据
- **动效适度**: 动效设计要适度，避免过度动画影响数据读取
- **实时反馈**: 提供明确的数据更新和状态变化反馈

## 3. 色彩方案

### 3.1 主色调

- **主色**: `#409EFF` (Element UI主题色)
  - 用于主要按钮、链接、高亮状态等
  - 表示系统的核心功能和操作

- **辅助色**: `#67C23A` (成功色)、`#E6A23C` (警告色)、`#F56C6C` (错误色)
  - 分别用于成功、警告、错误等状态提示

### 3.2 中性色

- **背景色**: `#F5F7FA` (浅灰)、`#FFFFFF` (白色)
  - 用于页面背景、卡片背景等

- **文字色**: `#303133` (主文字)、`#606266` (次要文字)、`#909399` (辅助文字)
  - 用于不同层级的文字信息

- **边框色**: `#E4E7ED` (边框)、`#EBEEF5` (分割线)
  - 用于组件边框、分割线等

### 3.3 大屏特殊色彩

- **大屏背景**: `linear-gradient(135deg, #0a1929 0%, #0b2133 50%, #0a1929 100%)`
  - 深色科技风格背景，营造专业感

- **大屏高亮**: `rgba(0, 255, 255, 0.3)`
  - 用于大屏标题、重点模块的发光效果

- **模块边框**: `1px solid rgba(255, 255, 255, 0.1)`
  - 半透明边框，增强层次感

- **模块背景**: `rgba(0, 0, 0, 0.3)`
  - 半透明背景，确保内容清晰可见

## 6. 布局规范

### 6.1 管理界面布局

- **整体布局**: 顶部导航栏 + 左侧菜单 + 主内容区
- **响应式设计**: 支持不同屏幕尺寸的适配

#### 6.1.1 空间规范

| 元素类型 | 间距规范 | 应用场景 |
|----------|----------|----------|
| 卡片间距 | 16px | 卡片之间的水平和垂直间距 |
| 组件间距 | 8-12px | 表单元素、按钮等组件之间的间距 |
| 区域间距 | 20px | 功能区域之间的间距 |
| 内边距 | 16px | 卡片、面板等容器的内边距 |

### 6.2 大屏布局规范

- **分辨率**: 标准分辨率为4800x1440px
- **布局结构**: 模块化布局，支持拖拽调整位置和大小

#### 6.2.1 模块布局规范

- **模块边框**: 1px 半透明白色边框
- **模块阴影**: 0 0 20px rgba(0, 255, 255, 0.2)
- **模块间距**: 最小间距为20px
- **模块标题**: 左上角，32px字号，白色，加粗

#### 6.2.2 各类型大屏布局模板

##### 6.2.2.1 日常版大屏(P1)布局

- **左侧区域**: 天气信息、限行尾号、工作规划三个模块
- **中间区域**: 党建园地、会员单位概况两个模块
- **右侧区域**: 品牌活动、行业统计两个模块
- **右上角**: 日期和温度信息

##### 6.2.2.2 行业版大屏(P4)布局

- **顶部**: 行业发展趋势大屏标题
- **第一行**: 城市运营里程排名、行业发展趋势、投资规模统计
- **第二行**: 主要经济指标、技术创新情况、政策环境分析
- **第三行**: 运营安全状况、环保效益分析、人才队伍建设

## 7. 组件规范

### 7.1 通用组件

#### 7.1.1 按钮

- **主要按钮**: 使用主色`#409EFF`，圆角4px
- **次要按钮**: 使用白色背景，主色边框
- **文字按钮**: 无背景无边框，仅显示文字
- **禁用按钮**: 使用灰色，透明度0.6

#### 7.1.2 输入框

- **普通输入框**: 1px边框，聚焦时边框变色并加阴影
- **搜索框**: 左侧带搜索图标
- **文本域**: 可调整高度，带字数统计

#### 7.1.3 表格

- **表头**: 浅灰色背景，加粗文字
- **行高**: 36px
- **斑马纹**: 奇数行白色，偶数行浅灰色
- **悬停效果**: 行背景色变浅灰色

#### 7.1.4 卡片

- **边框**: 1px 浅灰色边框
- **圆角**: 4px
- **阴影**: 轻微阴影效果
- **标题**: 左侧对齐，加粗

### 7.2 大屏专用组件

#### 7.2.3 数据指标卡

## 8. 交互设计

### 8.1 通用交互

- **悬停效果**: 按钮、卡片等元素在鼠标悬停时显示高亮效果
- **点击反馈**: 所有可点击元素都应有明确的点击反馈
- **加载状态**: 数据加载时显示加载动画
- **错误提示**: 操作失败时显示明确的错误信息

### 8.2 大屏特殊交互

#### 8.2.1 编辑模式交互

- **模块选中**: 选中模块时边框高亮，显示操作按钮
- **拖拽调整**: 支持拖拽调整模块位置和大小
- **双击编辑**: 双击模块进入内容编辑模式
- **属性面板**: 右侧或底部显示模块属性编辑面板

#### 8.2.2 预览模式交互

- **缩放控制**: 支持缩放预览大屏
- **平移控制**: 支持鼠标拖拽平移视图
- **实时更新**: 数据更新时有轻微的数字变化动画
- **紧急信息**: 紧急信息显示时有醒目的动画效果

## 9. 响应式设计

### 9.1 断点设置

| 断点 | 屏幕宽度 | 布局调整 |
|------|----------|----------|
| 超大屏 | ≥1600px | 完整布局，多列显示 |
| 大屏 | 1400-1599px | 适当调整列数和间距 |
| 中屏 | 1200-1399px | 减少列数，调整组件大小 |
| 小屏 | 1024-1199px | 进一步简化布局，组件垂直排列 |
| 平板 | 768-1023px | 单列布局，简化部分功能 |
| 移动端 | <768px | 完全响应式，仅显示核心功能 |

### 9.2 大屏适配方案

- **按比例缩放**: 根据显示设备分辨率按比例缩放
- **布局调整**: 在小分辨率下自动调整模块布局
- **内容优先级**: 根据屏幕大小显示最优先级的内容
- **手动调整**: 提供手动调整选项，满足特定场景需求

## 10. 动画效果

### 10.1 通用动画

- **淡入淡出**: 页面切换、模态框显示等场景
- **滑动效果**: 侧边栏展开/收起、抽屉式菜单等
- **加载动画**: 骨架屏、旋转图标等
- **反馈动画**: 按钮点击、表单验证等

### 10.2 大屏特殊动画

- **数据更新动画**: 数字变化时的平滑过渡效果
- **模块进入动画**: 大屏加载时的模块依次进入效果
- **紧急信息动画**: 紧急信息显示时的高亮闪烁效果
- **图表动画**: 图表数据更新时的平滑过渡效果

## 11. Figma设计规范

### 11.1 图层命名规范

- **组件命名**: `ComponentName_Variant`
- **页面命名**: `PageName_Section`
- **模块命名**: `Module_ModuleType`

### 11.2 组件结构规范

- **组件层级**: 遵循组件化设计原则，合理拆分组件
- **样式分离**: 使用设计令牌(Design Tokens)管理颜色、字体等样式
- **交互原型**: 为可交互元素添加交互原型

### 11.3 导出规范

- **格式**: SVG格式用于矢量图，PNG格式用于位图
- **命名**: 文件名与组件名保持一致
- **组织**: 按功能模块组织导出文件

## 12. 开发实现指南

### 12.1 技术栈

- **框架**: React (当前实现) / Vue 3 (推荐)
- **UI组件库**: Element UI
- **图标库**: FontAwesome
- **样式方案**: 内联样式 + CSS Modules

### 12.2 组件实现

- **优先使用原生组件**: 优先使用Element UI原生组件
- **自定义组件**: 自定义组件需符合设计规范
- **组件复用**: 提高组件复用率，避免重复开发
- **代码规范**: 遵循团队代码规范，添加适当注释

### 12.3 性能优化

- **懒加载**: 实现组件懒加载
- **虚拟化**: 长列表使用虚拟滚动
- **缓存**: 合理使用缓存机制
- **减少重绘**: 优化DOM操作，减少重绘和回流
