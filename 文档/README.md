## 模块概述

大屏管理模块是人民城轨管理系统的重要组成部分，提供以下核心功能：

1. **大屏列表管理**：集中管理所有大屏资源，支持搜索、筛选和排序
2. **大屏创建与编辑**：提供可视化编辑器，支持自定义大屏布局和内容
3. **大屏预览与控制**：实时预览大屏效果，控制大屏的启动和停止
4. **大屏运行监控**：监控大屏运行状态和性能指标
5. **多类型大屏支持**：支持日常版、协会版、行业版、城市版、科技版、紧急版等多种大屏类型
6. **紧急信息发布**：支持发布紧急信息到指定大屏

## 快速开始

1. **环境准备**：
   - 安装Node.js v14.x或更高版本
   - 安装npm或yarn
   - 安装Git

2. **代码获取**：
   ```bash
   git clone https://github.com/people-metro/people-metro-admin-v2.git
   cd people-metro-admin-v2
   ```

3. **安装依赖**：
   ```bash
   npm install
   ```

4. **开发运行**：
   ```bash
   npm start
   ```

5. **生产构建**：
   ```bash
   npm run build
   ```

6. **自动化部署**：
   ```bash
   chmod +x 文档/部署/大屏管理模块GIT部署脚本.sh
   ./文档/部署/大屏管理模块GIT部署脚本.sh
   ```
