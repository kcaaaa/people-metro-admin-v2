# 展会报名管理功能设计文档

## 1. 功能概述

展会报名管理功能是人民城轨2.0系统中展会管理模块的组成部分，旨在为展会组织者提供简单的第三方平台报名入口配置功能。该功能支持管理员配置第三方报名平台的入口信息，包括宣传图、名称、链接和开关状态，用户将通过点击此入口跳转到第三方平台完成报名流程。

## 2. 功能列表

| 功能模块 | 功能点 | 优先级 | 描述 |
|---------|--------|--------|------|
| 报名入口配置 | 宣传图上传 | 高 | 上传展会报名的宣传图片 |
|  | 报名名称设置 | 高 | 设置展会报名的名称 |
|  | 报名链接配置 | 高 | 配置指向第三方报名平台的链接 |
|  | 报名状态开关 | 高 | 控制报名入口的开启/关闭状态 |

## 3. 功能详细描述

### 3.1 报名入口配置

#### 3.1.1 功能描述

报名入口配置模块允许管理员配置指向第三方平台的报名入口信息，包括上传宣传图、设置名称、配置链接以及控制开关状态。

#### 3.1.2 功能结构

```
报名入口配置
├── 宣传图上传
├── 报名名称设置
├── 报名链接配置
└── 报名状态开关
```

#### 3.1.3 功能字段

| 字段名 | 类型 | 说明 | 必填 | 默认值 |
|--------|------|------|------|--------|
| 宣传图 | 文件 | 展会报名的宣传图片 | 否 | 无 |
| 报名名称 | 字符串 | 展会报名的显示名称 | 是 | "展会报名" |
| 报名链接 | 字符串 | 第三方报名平台的URL | 是 | 无 |
| 报名状态 | 布尔值 | true-开启报名，false-关闭报名 | 是 | false |

#### 3.1.4 交互逻辑

- 管理员可以上传宣传图片，支持常见图片格式（JPG、PNG等）
- 报名名称将显示在前端页面，用于标识此报名入口
- 报名链接必须是有效的URL格式，指向第三方报名平台
- 开启报名状态后，前端页面将显示报名入口，用户可点击进入第三方平台
- 关闭报名状态后，前端页面的报名入口将隐藏或显示为不可用状态
- 所有配置保存后即时生效

## 4. 数据流向图

```
管理员操作 → 配置界面 → 保存配置 → 更新数据库 → 前端展示
                                       ↓
                                  第三方报名平台
                                       ↓
                                  用户报名操作
```

## 5. 业务流程图

```
开始 → 管理员访问展会报名管理页面 → 配置报名信息（宣传图、名称、链接、状态） → 保存配置 → 配置生效 → 结束
```

## 6. 异常处理机制

| 异常情况 | 处理方式 |
|---------|--------|
| 图片上传失败 | 显示错误提示，要求重新上传 |
| 链接格式不正确 | 显示验证错误，要求输入有效的URL |
| 保存配置失败 | 显示系统错误提示，建议重试 |
| 第三方平台链接失效 | 前端显示提示信息，建议联系管理员 |
